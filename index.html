Informe Final de Servidores


Una empresa requiere configurar una plataforma Chamilo de la siguiente manera. Debe funcionar con la última versión estable de php por medio de apache y con el puerto 81 y acceder por medio de un proxy inverso a través de ngnix, además la base de datos debe estar en un servidor mariadb independiente. La solución debe funcionar por medio de https. Genere una guía montada en github pages, exponga la solución.

-Paso 1: Instalar Ubuntu en Virtual Box

Para Instalar ubuntu debemos descargar la iso, luego descargar virtualbox, una vez hecho esto abrimos virtualbox y seleccionamos la iso de ubuntu e iniciamos la maquina



-Paso 2: Instalar Apache y PHP

Una vez iniciada la maquina, instalamos apache con el siguiente comando

apt install apache2

Luego de esto se empezara a instalar apache




-Paso 3: Cambiar el puerto a 81

Sabemos que por defecto el puerto esta en el 80, pero ahora lo vamos a cambiar al 81, para eso digitamos el siguiente comando
sudo apt /etc/apache2/ports.conf



Luego provamos en el navegador que efectivamente este escuchando el puerto 81, para eso ponemos la ip de la maquina, dos punto y el puerto 81
192.168.1.203:81



-Paso 5: Instalacion de mariadb

Ejecutamos el siguiente comando para instalar MariaDB en nuestro sistema Ubuntu:
sudo apt install mariadb-server

MariaDB debería iniciarse automáticamente después de la instalación. Puedes verificar su estado con el siguiente comando: iniciar sesión en el shell de MariaDB con el siguiente comando:
sudo mysql -u root -p




-Paso 6: Creacion de la base de datos

CREATE DATABASE chamilodb
